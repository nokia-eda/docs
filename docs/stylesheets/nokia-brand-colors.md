# Nokia brand colors (reference)

This page is a **convenience extract** for doc and theme work. The canonical specification is [Nokia Brand Hub — Color](https://brand.nokia.com/guidelines/the-basics/color-gradients/color). It does not replace Brand Hub.

**Sources:**

- Theme data: [`data.json`](../reference/brand-hub-color/data.json) (mirror of [brand.nokia.com/.../data.json](https://brand.nokia.com/themes/custom/nbh/js/data.json)), which drives the on-page color wheels.
- Saved full-page HTML exports (Downloads): `Color _ Nokia Brand Hub.html` (Blue), `Color _ Nokia Brand Hub-teal.html`, `Color _ Nokia Brand Hub-green.html`, `Color _ Nokia Brand Hub yellow.html`, `Color _ Nokia Brand Hub orange.html`, `Color _ Nokia Brand Hub red.html`, `Color _ Nokia Brand Hub pink.html`, `Color _ Nokia Brand Hub-purple.html`, `Color _ Nokia Brand Hub grey.html` — used to verify SVG wheels where the save matched the selected primary (Teal/Green saves often still contained the default Blue shade SVG; ramps below come from `data.json`).

**Generated:** 2026-03-27

Shade order matches the Hub secondary wheel: **500, 600, 700, 800, 900, 100, 200, 300, 400** (clockwise from the top notch).
